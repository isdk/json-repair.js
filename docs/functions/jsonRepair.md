# Function: jsonRepair()

> **jsonRepair**(`input`, `schema?`, `options?`): `Promise`\<`any`\>

Defined in: [repair.ts:21](https://github.com/isdk/json-repair.js/blob/51ba9c939ae687b9da4b8574f2af0af0f31a3c0e/src/repair.ts#L21)

Repairs and parses a JSON string generated by an LLM, guided by a JSON Schema.
This is the main entry point for the library.

## Parameters

### input

`string`

The potentially broken JSON string to be repaired.

### schema?

`any`

Either a plain JSON Schema object or a pre-initialized SchemaWalker instance for better performance in batch processing.

### options?

[`RepairOptions`](../interfaces/RepairOptions.md) = `{}`

Optional configuration to tune the repair process.

## Returns

`Promise`\<`any`\>

A promise that resolves to the repaired and parsed JavaScript object.

## Example

```typescript
const schema = { type: 'object', properties: { name: { type: 'string' } } };
const result = await jsonRepair('{ name: John }', schema);
console.log(result); // { name: 'John' }
```
